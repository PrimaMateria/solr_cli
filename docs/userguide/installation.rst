First steps
===========

Installation
------------

To install solr_cli from Pypi: ::

  pip install mysolr


From source code: ::

  python setup.py install

Dependencies
............

solr_cli uses a python client called mysolr_ . So, if you install solr_cli from
source code you have to install it.


.. _mysolr: http://mysolr.redtuna.org
